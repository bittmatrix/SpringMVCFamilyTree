FamilyTreeMVC is project based on MVC structure. I used SpringMVC with Maven to develop this project by using Eclipse Java EE IDE for Web 
Developers. PersonRestConroller serves SpringMVC application to interact with database. PersonDAO uses JDBC to Perfom CRUD operation with 
MySql Database. In order To run this application you can import zip file in Eclipse IDE. You need to have Tomcat server to run this 
application.

Following Maven dependencies used for this project:
MySQL Connector/J
Spring Web MVC
jackson-databind
Java Servlet API
javax.servlet.jstl
